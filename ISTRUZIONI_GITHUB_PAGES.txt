PUBBLICARE IL CONFIGURATORE SAPUMO SU GITHUB PAGES

1. Vai su GitHub con il tuo account.
2. Clicca + in alto a destra > New repository.
3. Nome consigliato repository: sapumo-configuratore
4. Mettilo Public se vuoi usare GitHub Pages gratis senza complicazioni.
5. Crea il repository.
6. Dentro il repository clicca Add file > Upload files.
7. Trascina i file estratti da questo pacchetto, non lo ZIP.
8. Clicca Commit changes.
9. Vai su Settings > Pages.
10. In Build and deployment scegli Deploy from a branch.
11. Source: main / root.
12. Salva.

Il link sarà circa:
https://TUO-NOME-UTENTE.github.io/sapumo-configuratore/

Se il sito non appare subito, aspetta 2-5 minuti e ricarica.
