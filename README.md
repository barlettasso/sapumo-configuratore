# Sapumo Configuratore

Versione definitiva congelata del configuratore Sapumo.

## Contenuto

- `index.html`: configuratore completo, pronto per GitHub Pages.
- `.nojekyll`: evita che GitHub Pages applichi Jekyll.

## Pubblicazione rapida con GitHub Pages

1. Crea un nuovo repository GitHub, ad esempio `sapumo-configuratore`.
2. Carica nella root del repository questi file: `index.html`, `.nojekyll`, `README.md`.
3. Vai in `Settings > Pages`.
4. In `Build and deployment`, scegli `Deploy from a branch`.
5. Seleziona branch `main` e cartella `/root`.
6. Salva. Dopo qualche minuto GitHub genera il link pubblico.
